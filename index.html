Photoshop Layer Search

Quickly search through your photoshop layers (PS5). Focus by click.

Download

It is very hard to find a specific layer in photoshop. Photoshop CS5 doesn't support searching of layers. This plugin helps you to do so.

Quickly search through your photoshop layers. Focus by click.

Requirements

I have tested it only in Adobe PS5, Mac.

Screenshot

Install from source


git clone git@github.com:markupwand/photoshop-layersearch.git
cd photoshop-layersearch/
rake

How to use

After opening a psd file in photoshop, Window->Extensions->LayerSearch

FAQ

Extensions is grayed out for me


Go to Preferences-> Plugins.
Enable Additional Plugins Folder,
choose /Applications/Adobe Photoshop CS5/Plugins as the folder

Layers do not update when I open a new photoshop file, add layers.

This is just a first prototype. Just close and open the plugin again, it should work.
